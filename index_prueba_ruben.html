Frontend

"Lo que ves", se conoce como Frontend y es la parte del sitio web con la que interactúa el usuario, por eso ecucharás que esta de lado del cliente.

HTML

Es un lenguaje de marcado que se utiliza para el desarrollo de páginas de Internet.

CSS

(siglas en inglés de Cascading Style Sheets), en español "Hojas de estilo en cascada", es un lenguaje de diseño gráfico para definir y crear la presentación de un documento estructurado escrito en un lenguaje de marcado.

Framework

Significa entorno de trabajo​ o marco de trabajo​ y es un conjunto estandarizado de conceptos, prácticas y criterios para enfocar un tipo de problemática particular que sirve como referencia, para enfrentar y resolver nuevos problemas de índole similar.

Backend

"Lo que no ves", se conoce como Backend y es la que se conecta con el servidor y las bases de datos, por eso escucharas que esta de lado del servidor.

JavaScript

Es un lenguaje de programación interpretado, dialecto del estándar ECMAScript. Se define como orientado a objetos.

JQuery

jQuery es una biblioteca multiplataforma de JavaScript, que permite simplificar la manera de interactuar con los documentos HTML, manipular el árbol DOM, manejar eventos, desarrollar animaciones y agregar interacción con la técnica AJAX a páginas web.​

Lo que

no sabes

Código

el conjunto de instrucciones que se escriben en un editor de código para lograr crear una página web.

Desarrollador

Es la persona que tiene los conocimientos y habilidades para construir determinada aplicación, juego, o web, de la manera más segura posible, que sea fácil de usar y adaptable a las persona y que se pueda modificar o mejorar con el pasar del tiempo.

Editor de código

Un editor de código es programa informático que te permite escribir líneas de código para varios lenguajes de programación. Puedes abrir varios archivos a la vez y visualizar el código de otras aplicaciones.

Interfaz

La interfaz es la conexión funcional entre dos sistemas, programas o componentes, creando una zona de comunicación que permite el intercambio de información. De igual manera, cuando se habla de Sitios Web, se denomina interfaz al conjunto de elementos de la pantalla que permiten al usuario realizar.

Grid

Es una cuadrícula que nos permite crear estructuras en una página web que tiene filas: de izquierda a derecha y columnas de arriba a abajo son de 4 o 5.

Maquetación web

Es aquella parte gráfica o aquella idea que se tiene para una página web la cual también se le puede llamar layout y después se puede transformar en código con herramientas tales como HTML, CSS y JS para que los navegadores pueden interpretar y proyectar nuestras ideas.

Lenguaje de programación

Consiste en escribir una orden o dar una instruccion a una computadora el cual genera una consecuencia en un objeto, si pulsa unos botones que generan un código van ha hacer que un objeto vaya hacia donde tu quieras.

Programación

Escribir líneas de código para mejorar o realizar un proceso que se repite varias veces.

Programador

es aquella persona que transforma ideas en un código que la computadora pueda ejecutar para el desarrollo de software.

Programa

Es una secuencia de instrucciones que tu escribes para lograr una determinada tarea. En programación web; es como se verá tu página y la interacción que tendrá con tus visitantes. Es igual a una receta, pones primero una cosa, luego una pizca de la otra y al final, disfrutas de una deliciosa comida.

Responsive

También conocido como diseño adaptativo, es una técnica de diseño web que busca poder ver de forma adecuada una misma página en diferentes dispositivos. Desde ordenadores de escritorio a tablets y móviles.

Servidor

Se le dice así a una computadora conectada a una red informática que contiene datos, programas, etc., que dan servicio a otras computadoras a través de esta red. Es como una bodega de información que permanece en la Nube.

HTML

Es un lenguaje de marcado que se utiliza para el desarrollo de páginas de Internet. Se trata de la siglas que corresponden a HyperText Markup Language, es decir, Lenguaje de Marcas de Hipertexto, sirve exclusivamente para indicar cómo va ordenado el contenido de una página web.

Javascript

Es un lenguaje con muchas posibilidades, utilizado para crear pequeños programas que luego son insertados en una página web y en programas más grandes, orientados a objetos mucho más complejos. Con Javascript podemos crear diferentes efectos e interactuar con nuestros usuarios.

Lenguaje informático

se conoce como lenguaje de programación a un conjunto de instrucciones destinado a la construcción de otros programas informáticos. Su nombre se debe a que comprende un lenguaje formal que está diseñado para organizar algorismos y procesos lógicos que serán luego llevados a cabo por un ordenador o sistema informático, permitiendo controlar así su comportamiento físico, lógico y su comunicación con el usuario humano.
